# AI狼人杀

## 介绍
纯提示词驱动的ai狼人杀

## 特点
    - 提示词驱动，用户不需要干任何事，只需要给ai分配角色和配置大模型，然后看ai玩
    - 工具调用，ai会自动使用已有工具去干事，如狼人杀人、预言家查验、女巫救人、投票等
    - 上下文隔离，ai之间上下文隔离，只共享规则内的上下文，如狼人之间的上下文会共享，投票时会参考发言的上下文

## 开始

- 1、配置chat.js里的参数：
```text
// 大模型（聊天接口）代理地址，apikey和大模型配置
var url= "";
var apikey= "";
var model = ""
```

- 2、本地运行index.html
- 3、给AI分配角色然后开始
